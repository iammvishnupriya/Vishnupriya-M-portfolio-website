# Vishnupriya M - Portfolio Website

Modern, professional portfolio website showcasing my experience as a Java Full Stack Developer.

## Tech Stack
- React (Vite)
- Tailwind CSS
- Responsive Design

## Local Development

```bash
npm install
npm run dev
```

## Deploy to GitHub Pages

1. Push this code to a GitHub repository
2. Go to repository Settings â†’ Pages
3. Under "Build and deployment", select "GitHub Actions" as the source
4. Push to main branch - the site will automatically deploy

Your site will be available at: `https://your-username.github.io/portfolio-website/`

## Update Base URL

In `vite.config.js`, change the base to match your repository name:
```js
base: '/your-repo-name/',
```
